# LexiDownloaderTelegram
HOST your telegram bot for downloading files!

1)You will need to install youtube DLP and FFMPEG(INSTRUCTION BELOW)

You will need to extract envEXTRACT and populate your data(github protects from uploading .env)

2)a)Go to telegram Botfather: @BotFather
b)send him command: /newbot
c)create your bot and get your ID
d)launch lexi python bot
e)then when you'll host Lexi downloader, inside Lexi downloader send command /getMyUserID and then populate your user ID in     .env     to become Admin
